# Vikash Yadav Portfolio

Production-quality GitHub Pages portfolio built with React, TypeScript, Vite, Framer Motion and responsive CSS.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Create a GitHub repo named `belucky19.github.io`.
2. Push this project to the repo.
3. Go to Settings → Pages → Source → GitHub Actions or main branch `/docs` depending on your preference.
4. For manual deployment using gh-pages:

```bash
npm run build
npm run deploy
```

# Vikash Yadav Portfolio

Production-ready GitHub Pages portfolio for Vikash Yadav.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This repository includes GitHub Actions deployment in `.github/workflows/deploy.yml`.

Push to `main`, then check **Actions** and **Settings → Pages**.

Live URL:

```text
https://vikashyadavin.github.io
```
